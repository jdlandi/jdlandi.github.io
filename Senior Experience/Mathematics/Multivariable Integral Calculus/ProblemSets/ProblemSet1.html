Problem 3

Use Fubini's Theorem to evaluate: ∫02∫01x1+xydxdy

To start, lets go back to Fubini's theorem, what exactly is it? Well, it collects what we learned from Riemann Integration into the general double integral rule, but it also notes a big thing: We can change the order of integration. As in, whether or not we integrate by x or by y first, the end result is the same, however, this does not mean that the process of integration is as easy for both of them.

In our case, working through them, trying to integrate by x first is not as easy as using y, the reason for this being that with y, we can use simple integral log and u-substitution rules to integrate, whereas for x, we would have to delve into the world of regular subsitution, which is nowhere as easy.

Rearranging the integrals, we get:

∫01∫02x1+xydydx

Looking at the inside now, we notice that:

ddyln⁡|1+xy|=x1+xy

Therefore, ∫x1+xy=ln⁡|1+xy|, which we can once again subsitute back in.

[ln⁡|1+xy|]02=ln⁡|1+2x|−ln⁡|1+0|=ln⁡|1+2x|

Like clockwork, we plug this result back inside to solve again:

∫01ln⁡|1+2x|dx
[12((2x)ln⁡(2x)−2x)]01=[xln⁡(2x)−x]01

Now we come up an interesting situation, when we plug things in, we are left with:

ln⁡(2)−1−0ln⁡(0)

ln⁡(0) Is not definied, so we might expect that we cannot resolve our integral, however, due to l'hospital's rule and limits, we know that limx→0xln⁡(x) does exist and it is equal to 0. Simply substituting x for 2x in the natural log does not affect the limit and is still 0, so we can subsitute that back in and we get our final answer. ln⁡(2)−1

Problem 5

What region R in the xy-plane maximizes the value of ∬R(4−x2−2y2)dA ?

Well, going back to maximization, we know that in order to find a maximum, we must first take the derivative, then set that to 0 to find critical values. However, similar to how in two-dimensional calculus, we can just "strip" the integral away when taking the derivative, in three-dimensional calculus we can do the very same thing, or at least the idea is there.

What we see in this integral is the area underneath the plane 4−x2−2y2. However, here we can apply the fundamental theorem of calculus. The derivative of the area underneath a function is the same thing as the function itself because of the way that derivatives and integrals are seemingly "opposite" operations.

Translating to 3 dimensional calculus, we can say the same thing. Taking the derivative (or I guess gradient if you want in our case) results in the same as the function, the plane, the integrand.

Now, following the same old rules, we take the derivative, the plane, and set it to 0.

4−x2−2y2=0⇒y=±x2−42

Along this line, this is where we get a maximum. If we plug in any value that satisfies this equation into the initial integral, we get a value higher than any other. Because we are in 3-dimensions, it is very cool to see how our maximum does no longer have to just be a single point, but an entire line along a 3-dimensional surface.
