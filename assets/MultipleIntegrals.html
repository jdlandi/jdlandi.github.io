Double and Iterated Integrals over Rectangles

We can begin thinking about double integrals in a similar fashion we do with single integrals, but instead of a partitioning a line, we can partition a region R into small areas, ΔAk. We add up these tiny areas and as the width of the areas (Δyk and Δxk) approach an infinitely small value (a limit), we get the sum of the total area of R.

To find the value of this area, we simply pick any point within this area and take it's value. As we add up all these values, we approach the volume of the area bounded by R. This, for a non-limit approach of ΔAk is the Riemann approach to double integrals.

Double Integrals

  • A partition of R is simply one part of the general region R. In our example, every rectangle ΔAk forms a partition of R.

To begin our example, we want to first find the area of each of these small rectangles, but since we know their widths, we can say that for any general area, ΔA=ΔxΔy. For every k in the partitions Ak of R, we want to repeat this process. We can model this with a simple summation (A Riemann Sum for a double integral!):

Sn=∑k=1nf(xk,yk)ΔAk

This riemann sum represents taking the value of some point in the area (xk,yk) of the partition Ak, for very small changes of Ak, resulting in us adding all the points of f(x,y) in the region R. Now, we want to stop approximating, and get a more rigorous approach to this sum. This introduces the next idea:

  • The norm of a partition P is the largest width or length of any of it's rectangles, also denoted as ||P||. For example, if ||P||=0.1, then no rectangle in the partition has a width or length larger than 0.1.

We can now put this to use, by saying ||P||→0, we can infer that these rectangles, (Δxk,Δyk), approach infinitely small side-lengths. Using mathematical notation:

lim||P||→0∑k=1nf(xk,yk)ΔAk

But we can also note that as ||P||→0, the number of rectangles in total, n increases, so we can also write this limit as...

limn→∞∑k=1nf(xk,yk)ΔAk

This is the Riemann limit, Sn, which we can now denote using regular integrals. If the limit exists, then the f is integrable and the limit is called the double integral of f over R, written as:

∬Rf(x,y)ΔA or ∬Rf(x,y)ΔxΔy

Double Integrals over General Regions

Now we can move one from rectangular regions of R and start looking at what happens when R is some bounded region.

Double Integrals over Bounded, Nonrectangular Regions

One of the best parts about the Riemann sum is that like in R2 math, we can generalize the bounded area to a nonrectangular, smooth area. Simply speaking, regardless of whether or not the area we are integrating over is smooth or rectangular, we can still generalize the idea of integrating over planes to be able to integrate over these planes. This integral is called the double integral of f(x) over R.

Volumes

Now, in order to actually use this knowledge to get the volume of an object who's region is nonrectangular. We want to be able to take the idea of the iterated integral we defined before, and then generalize it a bit further.

We do this by changing the limits of integration. We still want to follow the idea of finding an area function, similar to the iterated integral idea, but instead of integrating over two set points, a and b, we want to integrate over two different points, g1(x) and g2(x), where these points are definined by the x coordinate of the current plane we are trying to integrate.

In short, we want to keep the idea of holding on coordinate constant while we integrate a plan on that coordinate, however, because the limits of integration are no longer rectangular coordinates, we need to find the function that defines this integral.

We can disect the equation shown under Figure 15.10 and see how it relates. The first potion, ∫g1(x)g2(x)f(x,y)dy is defining and integrating over this area. We are integrating over y while we hold x constant, but we want to integrate between two intervals, g1(x) and g2(x), both of which depend on x as that is what we are keeping constant.

Properties of Double Integrals

Like 2-dimensional integration, Double Integration has several algebraic properties that we can take advantage of.

  1. Constant Multiple: ∬Rcf(x,y)dA=c∬Rf(x,y)dA

  2. Sum and Difference:

    ∬R(f(x,y)±g(x,y))=∬Rf(x,y)dA±∬Rg(x,y)dA

  3. Domination:
    a. ∬Rf(x,y)dA≥0 if f(x,y)≥0 on R
    b. ∬Rf(x,y)dA≥∬Rg(x,y)dA if f(x,y)≥g(x,y) on R

  4. Additivity: If R is the union of two nonoverlapping regions R1 and R2, then

    ∬Rf(x,y)dA=∬R1f(x,y)dA+∬R2f(x,y)dA
